Démonstration 01 : Composant de démonstration

Voici le composant : Demo01Component. Il est composé de 3 fichiers :

  • demo01.component.ts : Fichier logique, apporte à angular la class du composant permettant de faire le lien avec l'html et le css
  • demo01.component.html : Fichier de structure HTML propre au component
  • demo01.component.css : Fichier de design CSS propre au component
