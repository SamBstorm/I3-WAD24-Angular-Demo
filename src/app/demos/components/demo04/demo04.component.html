{{title|uppercase}}

Suite à l'utilisation du pipe UpperCase, le titre : "{{title}}" est devenu : "{{title | uppercase}}".

Autre exemple avec la date

Sans pipe

Aujourd'hui, nous sommes le {{today}}.

Avec pipe (sans paramètre)

Aujourd'hui, nous sommes le {{today | date}}.

Avec pipe (avec paramètres)

Aujourd'hui, nous sommes le {{today | date : 'EEEE dd MMMM yyyy'}}.

Avec notre pipe custom : reverseText

Votre texte inversé est :

{{userInput | reverseText}}

{{userInput | reverseText | uppercase}}

Avec notre pipe custom : mathPow

Le carré de 2 est {{2 | mathPow}}

Le cube de 2 est {{2 | mathPow : 3}}

2 puissance 0 est {{2 | mathPow : 0}}
